## Troubleshooting
| Problem | Solution |
|--------------------------------|-------------------------------------------------------|
| Tool doesn't start | Run as Administrator and disable antivirus temporarily |
| KMS  failed | Check internet connection and port 1688 |
| SLMGR commands not working | Verify permissions and Windows version |
| Office  error | Reconfigure KMS host settings |
| Windows 11 compatibility error | Update Windows and run the setup guide again |
| Network connection issues | Disable firewall temporarily |
|  status not updating | Restart service and retry |
| Download / update failed | Use the latest release and check your internet connection |
